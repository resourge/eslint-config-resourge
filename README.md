# eslint-config-resourge

## Usage

Create file .eslintrc.cjs

```javascript

  extends: ['resourge']
};
```

### React

module.exports = {
  extends: ['resourge/react']
};
```

### React-native

module.exports = {
  extends: ['resourge/react-native']
};
```
